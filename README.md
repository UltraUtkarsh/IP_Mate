# IP_Mate
A very Simple tool which can be used to trace IP Addresses and websites.

# How to Install

1. git clone https://github.com/UltraUtkarsh/IP_Mate
2. cd IP_Mate
3. chmod +x *
4. bash setup.sh
5. bash inmate.sh

That's it.
